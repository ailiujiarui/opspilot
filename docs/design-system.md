# GridFlow 视觉规范

- 颜色：正文 `#20252b`，次要文字 `#78818a`，边框 `#d9dde1`，背景 `#f5f6f7`，主操作 `#2463eb`，成功 `#2f9d61`，警告 `#d19a1c`，错误 `#d94a58`。
- 字号：页面标题 24px，区块标题 13px，正文 12px，辅助说明 10 至 11px。
- 间距：基础单位 4px；工具栏 8px，面板 12 至 18px；控件圆角 4 至 6px。
- 状态：加载使用稳定占位，空数据提供清除操作，错误说明原因与恢复动作，只读模式禁用编辑，冲突状态要求重新加载。
- 图标：按钮优先使用 Lucide 图标；图标按钮必须有可访问名称和 tooltip。
- 密集场景：表头、单元格、状态栏按五级层次组织，焦点使用蓝色 2px 内描边，不仅依赖颜色传达错误。
