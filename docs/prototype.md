# GridFlow 低保真原型

## 桌面布局

```text
┌ 顶部应用栏：品牌 / 工作区 / 保存状态 / 账户 ┐
├ 页面标题：数据表名称 / 撤销重做 / 新建 ┤
├ 工具栏：搜索 / 状态筛选 / 排序 / 字段 ┤
├ # │ 任务 │ 负责人 │ 状态 │ 优先级 │ 日期 │ 金额 ┤
│   │ 可滚动虚拟数据区                         │
└ 记录数 / 渲染提示 / 更新时间                 ┘
```

目标尺寸：1366x768 与 1920x1080。窄屏只承诺浏览、搜索和筛选，复杂编辑保持横向滚动。
